# @giselles-ai/agent

Minimal CLI for creating and managing agents.

## Usage

```bash
npx @giselles-ai/agent create
npx @giselles-ai/agent add-skill <path>
npx @giselles-ai/agent add-hosted-skill <slug>
npx @giselles-ai/agent edit-setup-script
npx @giselles-ai/agent build

# or after global install
giselle create
giselle add-skill <path>
giselle add-hosted-skill <slug>
giselle edit-setup-script
giselle build
```
