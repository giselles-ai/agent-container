# Fertile Ground — マーケティングページのデザインコンセプト

## コンセプト

**Sandboxは檻ではなく、土壌である。**

Giselleの血脈は「オーケストレーション」——複数のAIが分業・連携して成果物を生む。Agent Sandboxはその延長で、オーケストレーションに"実行"を与えるランタイム。

このページはそのアイデンティティを「Fertile Ground（肥沃な土壌）」というメタファーで表現する。哲学のリゾーム（中心がなく、どこからでも接続し、横に広がり、生成する構造）を参照している。

## メタファーの体系

すべての表現はこのメタファーから必然的に導出される。項目ごとの「正しさ」ではなく、Fertile Groundから導かれる必然としての表現であること。

- **土壌 (Ground/Soil)**: ランタイム実行環境そのもの。エージェントが根を張る場所
- **膜 (Membrane)**: 隔離の境界。壁ではなく、必要なものを通し外の環境と浸透的につながる生きた境界面
- **リゾーム (Rhizome)**: Multi-agentの接続。中心がなく横に広がるネットワーク。ノードの接続線は直線ではなく有機的に伸びる
- **Sow / Tend / Harvest**: Build / Verify / Embed の農耕的な言い換え。種を蒔き、芽を確かめ、実を収穫する

## 言語のルール

- **生成の動詞を使う**: grow, take root, yield, bear fruit, emerge
- **制御の動詞を使わない**: isolate, restrict, contain, lock, cage
- ターミナル風の表現（コードブロックのアニメーション、ASCIIアート等）はGiselleの表現ではない。避けること
- 英語コピーは短く、動詞始まり。説明的な長文は避ける
- トーンは「Composed confidence」——「すごいでしょ」ではなく「当然こうなる」という静かな確信

## カラー

ダークでもライトでもない中間。朝の光のような温度感。

| トークン名 | 値 | 意味 |
|---|---|---|
| `ground` | `#f5f0e8` | 背景。土壌の表面 |
| `ground-deep` | `#ebe4d8` | カードや交互セクションの背景。少し深い土壌 |
| `soil` | `#2c2416` | テキスト。真っ黒ではない暖かい茶 |
| `soil-muted` | `#7a6e5c` | 副次テキスト |
| `membrane` | `#c9a87c` | ボーダー、アクセント。膜の色 |
| `membrane-light` | `#e2d2b8` | 薄い膜。hover状態や区切り線 |
| `root` | `#a08660` | ラベル、セクション名。根の色 |
| `sprout` | `#8a9a6c` | 成長の表現に使う緑（控えめ） |

## タイポグラフィ

- **見出し**: Instrument Serif — 温もりと確信を持つセリフ体
- **本文**: DM Sans — 暖かみのあるヒューマニストサンセリフ
- 見出しは大きく、余白をたっぷり。言葉の力で持たせる

## ビジュアル

- リゾームSVG: 有機的なベジエ曲線の根のネットワーク。直線なし
- 幾何学的なグリッドやターミナルUIではなく、有機的な形状
- 「1万行→10行」の語り方: 土壌が豊かなら、種一粒で済む
