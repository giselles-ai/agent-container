# Weekly report request

Create a concise weekly report for the Revra team.

## Required outputs
- Write the main report to ./artifacts/report.md
- Write a machine-readable summary to ./artifacts/highlights.json

## Audience
- CEO wants a fast status read
- Sales lead wants pipeline movement
- Support lead wants customer risk signals

## Report structure
1. Executive summary
2. Revenue and pipeline
3. Customer changes
4. Risks and follow-ups

Keep the tone direct and specific.
